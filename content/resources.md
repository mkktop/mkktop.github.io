## Linux

### 常用命令速查

```bash
# 查看系统信息
uname -a

# 磁盘使用
df -h

# 查看端口占用
ss -tlnp

# 实时日志
journalctl -f

# 查找文件
find / -name "*.conf" 2>/dev/null
```
